# Практична робота 11 — Класи та ООП

## Реалізовано

- Animal
- Mammal
- Bird
- Fish
- Zoo

## Використані технології

- JavaScript
- ES6 Classes
- Inheritance
- OOP

## Запуск проєкту

```bash
node index.js


Опис

У проєкті реалізовано систему зоопарку з використанням класів та принципів ООП.

Було створено:

базовий клас Animal;
класи Mammal, Bird та Fish;
наслідування через extends;
методи makeSound(), eat(), sleep(), move();
клас Zoo для управління тваринами.

Також використано:

constructors;
inheritance;
polymorphism;
encapsulation.
